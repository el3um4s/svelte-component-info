SvelteInfo

A Svelte Component to show info about Svelte Components
How to use
To import the package in a project:
npm i -D @el3um4s/svelte-info
To use in a file:
<script>

import { SvelteInfo } from "@el3um4s/svelte-info"

</script>
<SvelteInfo
  • name
  • description
  • info
  • urlPackage
> </SvelteInfo>
